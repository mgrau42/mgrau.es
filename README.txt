Buenos días.
Este repositorio contiene algunas de las funciones que he creado en la escuela de programación 42 telefónica.
Siguen la norma de la escuela, entre ellas no usar ciertas funciones o no inicializar las variables en la misma línea de la declaración.
Os invito a observar cómo estamos trabajando en 42 a través de mis proyectos, las funciones están comentadas y existe un README dentro de cada proyecto explicando el objetivo del mismo.

En un futuro planeo seguir actualizando este repositorio con proyectos en distintos lenguajes según vaya progresando dentro de la escuela.

Hasta entonces, gracias por su interés.

23/sep/2020
Añadida Libft, librería de uso general con funciones útiles para el resto del curso 42 Madrid

03/oct/2020
Añadido Get Next Line, proyecto cuyo objetivo era realizar una función que leyera línea a línea un texto pasado a través del fd.
Se trata de un buen ejercicio para entender las variables estáticas.

14/11/2020
Añadido ft_printf, proyecto que recrea la función printf de las librerías estándar.
Añadido ft_server, proyecto cuyo objetivo es crear un servidor con Nginx usando un único contenedor de docker.
